Sinew: Simple R Package Documentation

Chapter 1 Installation

1.1 CRAN

The CRAN package homepage

install.packages('sinew')

For a quick example, run the following:

sinew::makeOxygen('lm')

1.2 Development

The package is currently under continuous development for the most up to date version run the script below.

The github package homepage

remotes::install_github("metrumresearchgroup/sinew")
